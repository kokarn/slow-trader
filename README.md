# slow-trader

Configuration lives in environment.

## Config vars
`AVANZA_USERNAME`  
string  
the username to login with

`AVANZA_PASSWORD`  
string  
the password to login with

`AVANZA_TOTP_SECRET`  
string  
avanza totp secret configured by [the totp setup](https://github.com/fhqvst/avanza#getting-a-totp-secret)
